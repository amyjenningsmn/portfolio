Nile of the North

Interactive map of North Minneapolis displaying food related resources by category.

Users have the ability to create their own resources and associated content. Shared knowledge of these local resources encourages connection and use within the community.

Built as a team project while studying Full Stack Development at Prime Digital Academy in Bloomington, MN. Key technologies used: PostgreSQL, Express.js, AngularJS, Node.js, and Angular Material among others.

Prompted

Built as my solo project while at Prime Digital Academy using the MEAN stack, Bootstrap, Summernote, and the "I Need a Prompt" API.

Users are able to generate prompts, write in a rich-text environment, and save to a personal library, all in one application.
